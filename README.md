# 📊 Excel-Basic-to-Advanced-Projects
📊 Excel Projects Portfolio – A collection of 5 projects showcasing skills from basic formulas to advanced dashboards using Excel, Pivot Tables, Power Query and Power Pivot. Includes Payroll System, Advanced Charts, Student Marksheet, and Interactive Sales Dashboards.
 
This repository contains **5 Excel projects** ranging from basic formulas to advanced dashboards.  
During these projects, I made use of:  
- **Basic to Advanced Excel**  
- **Pivot Tables**  
- **Power Query & Power Pivot** (specially for Dashboards)  

---

## 📂 Projects Included  

1. **Employee Payroll System**  
   - Built using **data aggregation formulas** for salary calculation.  
   - Focuses on employee payroll automation.  

2. **10 Advanced & Tricky Charts**  
   - Created different types of **custom Excel charts**:  
     - Multi Category Chart  
     - Speedometer Chart  
     - Thermometer Chart  
     - Milestone Chart  
     - Waterfall Chart  
     - Gantt Chart  
     - Chart with Trend Arrow  
     - Chart with Actual & Target Value  
     - Data in Scatter Chart  
     - Dynamic Target Line on Bar Chart  

3. **Basic Sales Dashboard**  
   - A simple and interactive sales dashboard for tracking key metrics.  
   - Built with **Pivot Tables & Power Query**.  

4. **Student Printable Marksheet**  
   - Automated **marksheet generator** for students.  
   - Printable and user-friendly.  

5. **Advanced Sales Dashboard**  
   - Contains multiple dashboards:  
     - Primary Dashboard  
     - Product-wise Analysis  
     - Salesman-wise Analysis  
   - Created using **Power Query, Power Pivot & Pivot Tables** for automation.
